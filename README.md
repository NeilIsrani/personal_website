# Portfolio Website

A Next.js portfolio website for Neil Israni.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Pages

- **Home** (`/`) - Main page with name and navigation buttons
- **Professional** (`/professional`) - Professional portfolio page
- **Personal** (`/personal`) - Personal page

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules
